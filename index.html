Brief Abstract
The Global Navigation Satellite System (GNSS) is recognized as a prevalent method for determining user position. However, the GNSS result can be seriously biased in urban canyons, where the presence of civil constructions can result in multipath effects and affect positioning accuracy.

In this research, I use MATLAB to extract and manually label the corresponding navigation information, such as correlations, from raw satellite data collected from Software-Defined Radio (SDR) receivers in Hong Kong. I then use Python to construct a deep learning model and train it with previously labeled data to improve positioning accuracy in challenging multipath environments. The feasibility of the trained algorithm is evaluated by comparing the analysis results of the collected dataset with the conventional methods. The primary objective of this project is to incorporate a machine learning algorithm to optimize the accuracy of GNSS positioning in the receiver phase. I anticipate that the adequately trained deep learning algorithm will provide optimized results in identifying the userâ€™s precise position compared to the conventional algorithm. This research holds the optimistic potential to improve the positioning accuracy of GNSS receivers in an innovative and sustainable way with the integration of machine learning into the receiver processing phase.
